# Fake News Detector

A mobile Android application that helps users verify news and detect potential misinformation.

## About
Fake News Detector allows users to input any text or news article and receive an analysis of how credible the information might be. The goal is to help people identify fake news faster and develop more critical thinking when consuming information online.

## Tech Stack
- **Platform:** Android
- **Language:** Kotlin
- **IDE:** Android Studio
- **UI Framework:** Jetpack Compose
- **Database:** Room Database
- **AI Analysis:** Claude AI API

## Features
- Text input for news analysis
- AI-powered credibility assessment
- Save previous analyses locally
